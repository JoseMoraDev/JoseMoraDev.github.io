ETIQUETAS HTML EN UNA PÁGINA WEB

JERARQUÍAS DE TÍTULOS

<*h1>Esto es un título 1<*/h1>

<*h2>Esto es un título 2<*/h2>

<*h3>Esto es un título 3<*/h3>

<*h4>Esto es un título 4<*/h4>

<*h5>Esto es un título 5<*/h5>
<*h6>Esto es un título 6<*/h6>

El h1 es el título mas importante, solo hay que utilizarlo para los títulos principales

El h2 se debe de utilizar para los títulos normales

El h3 se utiliza para los títulos menos importantes

Y así sucesivamente hasta el h6

Nota importante: Aunque cada etiqueta h tiene un formato (tipo, tamaño de fuente...) predeterminado, no hay que utilizar estas etiquetas <*hnumero> para resaltar y/o dar formato al texto. Solo deben emplearse para indicar la jerarquía de títulos.

INSERTAR IMÁGENES

¿Cual es la manera de insertar una imagen como esta en nuestra web?

El código que se ha utilizado es: <*img src="hcabecera.jpg" alt="La imagen no se ha podido cargar" title="Cabecera HTML" width="1000px" height="270px";>

Expliquemos un poco el código:

--> <*img> es la etiqueta que empleamos para decirle al navegador que vamos a insertar una imagen. Es una etiqueta vacía, por lo que no requiere etiqueta de cierre.

--> src es el atributo donde especificamos la ruta de la imagen a mostrar.

--> alt es el texto alternativo que solo se mostrará si la imagen no se ha podido cargar

--> title me permite asignar un título a un elemento, que se mostrará cuando el usuario pase por encima el ratón

--> width indica el ancho de la imagen en píxeles si se pone px después del número, o en porcentaje si se pone 25% por ejemplo



Según la url o direccion en la que se ubique la imagen, distinguimos tres tipos de inserción:

INSERCION ABSOLUTA

Dentro del atributo src de la etiqueta imagen indicamos la dirección url externa en la que se encuentra la misma:

Por ejemplo: <*img src="http://4.bp.blogspot.com/-HkHoqQuxUQE/UXmuXLTl0UI/AAAAAAAAiyY/VYsiOVFBPQg/s1600/the-end.jpg">

INSERCION RELATIVA

La imagen se encuentra dentro del directorio en el que tenemos nuestra web, dentro de una de las carpetas del mismo.

Veamos un ejemplo: <*img src="carpeta/hcabecera.jpg">

INSERCION DIRECTA

En este caso, la imagen se encuentra en el directorio de trabajo, pero en la misma carpeta que la página, por lo que solo neceitamos indicar el nombre y extensión de la imagen:

Ejemplo de inserción directa: <*img src="html.gif">

FORMATO DEL TEXTO

ATRIBUTO STYLE Y SUS PARÁMETROS

Se utiliza el atributo style sobre un párrafo <*p> o un título <*h1> o otros elementos de texto

el atributo style permite modificar parámetros de la fuente como el color, tamaño, tipo de letra, o la alineación del texto


COLOR DEL TEXTO

Para cambiar el color de un elemento, por ejemplo de un párrafo se hace de la siguiente manera

<*p style="color:blue";> Este es un párrafo de ejemplo en color azul <*/p>


TAMAÑO DE LA FUENTE

se utiliza el parámetro font-size del atributo style: Se puede indicar el tamaño en píxeles o en porcentaje relativo:

<*p style="font-size:30px;"> Ejemplo de tamaño de fuente a 30 píxeles <*/p>

<*p style="font-size:150%;"> Ejemplo de tamaño de fuente al 150% <*/p>


TIPOGRAFÍA DE LETRA

Cambiamos el tipo de letra utilizando el parámetro font-family del atributo style, por ejemplo:

<*p style="font-family:verdana;">El tipo de fuente empleado en este párrafo es Verdana<*/p>


ALINEACIÓN DEL TEXTO

Para alinear un párrrafo utilizamos el parámetro text-align del atributo style. Text-align me permite elegir distintas opciones, las cuales son:

<*p style="text-align:left;">Alinear el párrafo a la izquierda<*/p>

<*p style="text-align:center;">Alinear el párrafo al centro<*/p>

<*p style="text-align:right;">Alinear el párrafo a la derecha<*/p>

COLOR DE FONDO DE LA PÁGINA

Si utilizamos el parámetro background-color del atributo style sobre la etiqueta body, es posible cambiar el color de fondo de toda la páquina

Veamos el ejemplo aplicado en esta web: <*body style="background-color:lightgray;">
